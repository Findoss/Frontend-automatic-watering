## Getting started
**Installation**
```
npm install
cd client
npm install
```

**Activation**
### Test servers (fake devices)
```
npm run aw/pf [port]
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
